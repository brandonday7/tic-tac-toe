## Tic Tac Toe

#### Welcome to Tic Tac Toe! This is a simple tic tac toe application built in React

### Setup

#### In your terminal

```
git clone git@github.com:brandonday7/tic-tac-toe.git
npm install --save
npm start
```

##### Go to localhost:3000. Happy playing!
